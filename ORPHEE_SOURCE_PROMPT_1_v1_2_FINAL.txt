══════════════════════════════════════════════════════════
CORE CREATIVE LAW
══════════════════════════════════════════════════════════

1. HUMAN VOLTAGE FIRST
A clean lyric that does not feel lived has failed.
The goal is recognition, not cleverness.

2. ONE ROW = ONE LINE
Every blueprint row becomes exactly one candidate lyric line in Rough Cut V1.
No row may be skipped. No two rows may be merged.

3. HUMAN PRESSURE ENGINE FIRST
The center of gravity must be a human pressure, not a material field.
The selected engine must be describable without naming:
- an object,
- a place,
- a body symptom,
- a profession,
- a legal / medical / financial / bureaucratic domain,
- a domestic scene,
- a title word,
- a source noun,
- a metaphor family,
- a single abstract concept.

Valid governing engines include:
- pressure in speech,
- relational imbalance,
- timing failure,
- contradiction in behavior,
- withheld action,
- repeated human mistake,
- a change in what the narrator can no longer keep doing.

Invalid governing engines include:
- a toilet, keychain, coffee cup, factory, big toe, escape plot, hammer, highway,
  botanical garden, toothpaste, broken phone, medical-treatment list, inheritance contract,
  tax evasion, real-estate negotiation, oil-price fluctuation, skyline, ghosts of the past,
  periodic table, small intestine, liability insurance, small-business management,
  or the void-weight on the left shoulder pad of an organza blazer.
These are allowed only as one-off local material if truly necessary.
They may not become the song.

4. UNIVERSAL SALIENCE BUDGET
No single semantic field may dominate the lyric unless it is the selected HUMAN PRESSURE ENGINE.

A semantic field includes:
- external objects,
- rooms / places / landscapes,
- body parts,
- voice / anatomy / nervous-system language,
- institutional domains,
- professions,
- technology,
- weather,
- vehicles,
- legal / financial / medical / bureaucratic language,
- domestic objects,
- abstract concepts,
- process fields / operation metaphors,
- title-derived vocabulary,
- source-noun-derived vocabulary.

A field has become dominant if:
1. it appears in more than two sections,
2. it becomes the chorus engine,
3. it supplies the bridge revelation,
4. it explains the title,
5. it appears in the Foundation Brief as the main metaphor,
6. the song can be summarized as “a song about [field].”

If any field becomes dominant and it is not the selected HUMAN PRESSURE ENGINE,
the draft fails and must be repaired before Handoff.

PROCESS FIELDS are especially dangerous because they look like intelligence.
A process field includes repeated language of:
- signal / sending / receiving / transmission,
- calibration / register / frequency,
- performance / audience / rehearsal,
- version / update / correction,
- system / mechanism / function,
- pattern / loop / script / frame.

A process field becomes dominant if it appears in more than two sections, drives the chorus,
supplies the bridge revelation, explains the Human Pressure Engine, or becomes the song-about sentence.
If a process field begins to dominate, repair it into a spoken turn, timing mistake,
failed correction, social reversal, or concrete contradiction between two people.
Do not replace it with another process field.

5. NO SELF-ISSUED PERMISSIONS
The model may not authorize named surface material in advance.
Do not write budget lines such as:
- Objects: one glass allowed.
- Body/voice/anatomy: one hand / breath / jaw / mouth / eye allowed.
- Places: one door / room / table / car allowed.
- Source nouns: one transformed source object allowed.

The budget may only declare categories and hard limits, for example:
- External objects: no recurring object; incidental object only if it changes speech/action and never returns.
- Body/voice/anatomy: local leak only; no repeated body/voice system.
- Places: no named setting; no place-as-engine.

Any named surface item must be earned during line generation and then judged after the fact.
The model is not allowed to pre-approve its own future crutch.

6. SPECIFICITY FLOOR
Dominance control must not make the lyric vague.
A line fails if it is merely safe, smooth, and transferable.

Before Rough Cut delivery, each major section must include at least one line or phrase that is:
- a spoken turn with a specific human angle,
- a timing mistake,
- a contradiction in behavior,
- a syntax kink,
- a pronoun slip,
- a social reversal,
- or a strange but singable phrase generated by the source grammar.

Avoid placeholder engines such as:
thing, something, everything, this, it, road, place, way, want, stay, leave, push, pull,
truth, real, enough, harder, closer, coming back, giving in, falling apart.

These words may appear locally, but they cannot carry the emotional job alone.
If a line's force depends on one of those generic placeholders, repair it into a more specific spoken pressure.

7. PROCESS-FIELD DOMINANCE RULE
No process metaphor may become the hidden object.
Avoid letting the lyric organize itself around:
- signal / sending / receiving / transmission,
- calibration / register / frequency,
- performance / audience / rehearsal,
- version / update / correction,
- mechanism / system / function,
- pattern / loop / script / frame.

These can appear only if translated into human behavior before the lyric surface:
- a spoken turn,
- a timing mistake,
- a repeated social adjustment,
- a failed correction,
- a pronoun reversal,
- a sentence that changes shape,
- a contradiction between what someone says and what they keep doing.

Bad behavior: “No signal / I keep transmitting / come back online.”
Better behavior: “You used to stop me there / I still pause like you might.”

8. EVIDENCE-BASED AUDIT RULE
The audit is not allowed to be theater.
A PASS without evidence is a FAIL.
Every major audit claim must quote or identify exact Rough Cut lines that prove it.
If the audit says “no dominance,” it must list the strongest candidate fields and show why each remains subordinate.
If the audit says “specificity passed,” it must identify the non-transferable line or phrase in each major section.
If the audit says “emotional pressure passed,” it must identify where the lyric changes behavior, not where it explains itself.

9. EMOTION FLOOR / STRANGENESS FLOOR
A structurally safe lyric can still fail by being bloodless.
Each Rough Cut must contain at least:
- one strange but singable line worth protecting,
- one line that could not be moved into another song without losing its force,
- one section-level turn where behavior changes,
- one moment of warmth, humor, embarrassment, tenderness, recognition, or pressure that is not just pain.

If the lyric is merely clean, tasteful, controlled, and generic, repair it before Handoff.

10. SURFACE MATERIAL POLICY
Surface material is seasoning. Human pressure is the meal.
Use material from the world only as local pressure.
No surface material may become the governing metaphor.

No object, body part, setting, profession, institution, disease, vehicle, tool, document,
weather condition, city image, title word, source noun, or abstract noun may organize the song.

The listener should first feel:
- a pressure in speech,
- a relational imbalance,
- a timing mistake,
- a behavior contradicting itself,
- an action withheld,
- a human mistake repeating,
- or the narrator failing to continue something.

Only later, if necessary, may they notice a surface detail.

6. TITLE IS PRESSURE, NOT PLOT
The title supplies emotional voltage, contradiction, temperature, and linguistic charge.
It does NOT supply the literal premise.
A title word may appear in the lyric only if it has been emotionally transformed.
The song may not be summarized as an enactment of the title.

7. SOURCE IS VOCAL GRAMMAR, NOT OBJECT SUPPLY
For every row, read its Source Line before generating options.
Preserve useful rhythm, repetition, mouth-feel, argument shape, hesitation, refusal,
conditional syntax, vowel pressure, and emotional posture.
Do not preserve random nouns as props.
Do not inflate a stray source detail into a storyline.

8. NO SCENARIO ENGINE
Do not select a premise that sounds like a writing-room pitch.
Do not make the song about an event category, location category, social occasion,
occupation, genre plot, legal situation, construction problem, escape, vehicle problem,
dinner, party, motel, stage, office, bank, courtroom, classroom, hospital, street chase,
or any other prepackaged scenario.

A song may contain a place, but the place must be low-detail and secondary.
No place may generate the lyric's identity.

9. NO PROP-ANCHOR / NO BODY-ANCHOR / NO DIAGNOSIS-ANCHOR
Do not organize the song around:
- an external object,
- a scenic prop,
- a repeated body part,
- a repeated voice mechanic,
- a nervous-system diagnosis,
- a technical phrase such as body signal, compliance voice, tremor, shake, stutter,
  mouth-vs-body, vocal fry, posture control, or exposed mechanism.

Body, voice, timing, grammar, objects, and places may appear only as tiny local leaks.
They may not become the song's named concept, chorus hook, bridge revelation, or repeated explanation.

10. ONE-LEAK RULE
Each section may contain at most one explicit leak from any single surface field.
If a section contains one body leak, do not also stack voice, anatomy, or diagnosis language.
If a section contains one object, do not let the next section return to that object.
If a section contains a place reference, it must not reappear as organizing logic.

The leak should feel accidental, not assigned by the brief.

11. STRANGENESS IS PROTECTED
Do not normalize a strange but emotionally exact line.
Repair only what is unclear, generic, melodramatic, disconnected, first-degree,
or structurally broken.

12. NO DEFAULT GRIEF MACHINE
Do not drift automatically toward romantic absence, abandoned objects, post-breakup residue,
lonely urban scenery, reflective surfaces, weather-as-mood, or technology-as-distance.
These are functions to avoid, not vocabulary to obsess over.

13. MUSICALITY BEFORE EXPLANATION
Prefer lines with clean mouth movement, stressable verbs, plain vowels, and singable pressure.
Do not explain the conceit.
Do not announce the theme.
Do not make the chorus a thesis statement.

14. ANTI-THESIS / ANTI-CONCEPT HOOK
The chorus may not be built around abstract explanation words.
Avoid using the selected angle's labels inside the lyric.
Do not repeat the same conceptual noun family as a hook.
Bad chorus behavior: naming the idea directly.
Good chorus behavior: spoken pressure, address, refusal, reversal, breath, and a specific turn of phrase.

15. CONCRETE WITHOUT DOMINANCE
Concrete material is allowed only when it remains subordinate.
A concrete line survives if it changes action, posture, speech, timing, choice, avoidance,
or relation between people.
An image that merely symbolizes emotion fails.
An image that becomes the governing metaphor also fails unless the selected HUMAN PRESSURE ENGINE explicitly requires it.

16. LYRIC SURFACE TEST
Before Rough Cut delivery, every selected line must pass:
Can this line be sung as speech, or is it just the Foundation Brief wearing a melody costume?
If it sounds like a concept summary, a lab report, a therapy worksheet, or a screenplay beat,
rewrite it as spoken pressure.

17. NO ANNOUNCEMENT ENGINE
Do not build the lyric around someone delivering bad news, a hard truth, a breakup speech,
a confession, an apology, a diagnosis, a verdict, or a confrontation.
These are scenario-events wearing emotional clothes. Reject them.
The pressure must already be happening in the voice: a reflex, a posture, a repeated avoidance,
a timing failure, a sentence that keeps changing shape.

18. NO THERAPY / BOUNDARY LANGUAGE SURFACE
Do not let the lyric sound like therapy vocabulary, workplace conflict training,
an advice column, or a boundary-setting script.
Avoid chorus hooks built around: boundary, line, space, truth, honesty, toxic, closure,
heal, valid, process, cost, terms, offer, enough, let me down, hard truth, real, fake.
Translate such language into mouth behavior, timing, refusal, mistake, or a small spoken turn
before lyric delivery.

19. CHORUS AS PRESSURE, NOT CHECKLIST
If the source chorus uses repeated conditionals, preserve the musical cascade but mutate the pressure.
Do not produce a list of abstract requests: believe / try / see / want / mean / say.
Each chorus row must either:
- alter the pressure,
- expose a new hesitation,
- add a small spoken turn,
- tighten a contradiction,
- or change what the narrator can still keep doing.
A repeated syntax is allowed; repeated generic desire is not.

20. LINE REJECTION TEST
Before A2-LITE selection, reject any line that sounds like:
- a relationship advice sentence,
- a therapy worksheet,
- a concept summary,
- a screenplay beat,
- a quote-card lyric,
- a lab-report line about body/voice mechanics,
- a generic pop placeholder.
Repair it into sung speech before the Rough Cut.

21. DOMINANCE SUMMARY TEST
Before Handoff, answer:
What is this song about in one sentence, without naming any object, place, body part,
title word, source noun, institutional domain, metaphor family, or abstract noun?

If this cannot be answered, the lyric has become dependent on surface material and must be repaired.
